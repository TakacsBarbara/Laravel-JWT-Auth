Introduction

Laravel, powered by JWT authentication, ensures a secure and efficient user authentication process.

Base URL: http://localhost

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

To authenticate requests, include an Authorization header with the value "Bearer {YOUR_AUTH_KEY}".

All authenticated endpoints are marked with a requires authentication badge in the documentation below.

You can retrieve your token by visiting your dashboard and clicking Generate API token.

Authentication

Register a new user with some datas.

Example request:
curl --request POST \
    "http://localhost/api/register" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"name\": \"quis\",
    \"email\": \"fermin.berge@example.org\",
    \"password\": \"&J;~W}3s\",
    \"phone_number\": \"fugiat\",
    \"address\": \"accusamus\",
    \"position\": \"nesciunt\"
}"

const url = new URL(
    "http://localhost/api/register"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "name": "quis",
    "email": "fermin.berge@example.org",
    "password": "&J;~W}3s",
    "phone_number": "fugiat",
    "address": "accusamus",
    "position": "nesciunt"
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());

Example response (201):


{
    "user": {
        "name": "John Doe",
        "email": "johndoe@example.com",
        "phone_number": "1234567890",
        "address": "123 Main St",
        "position": "Developer"
    },
    "msg": "User created successfully"
}

Example response (400):


{
    "error": {
        "email": [
            "The email field must be unique."
        ],
        "password": [

Request      

POST api/register

Headers

Content-Type      

Example: application/json

Accept      

Example: application/json

Body Parameters

name   string   

The name of the user. Example: quis

email   email   

The email address of the user (must be unique). Example: fermin.berge@example.org

password   string  optional  

required|min:8|regex:/^(?=.[A-Z])(?=.\d).+$/ The user's password. Example: &J;~W}3s

phone_number   string  optional  

optional The phone number of the user. Example: fugiat

address   string  optional  

optional The address of the user. Example: accusamus

position   string  optional  

optional The position of the user. Example: nesciunt

Log in a registered user with email and password.

Example request:
curl --request POST \
    "http://localhost/api/login" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"email\": \"lebsack.maddison@example.org\",
    \"password\": \"i|l$lHwnijs,.\\\"OJd.w\"
}"

const url = new URL(
    "http://localhost/api/login"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "email": "lebsack.maddison@example.org",
    "password": "i|l$lHwnijs,.\"OJd.w"
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());

    --header "Accept: application/json"

);
